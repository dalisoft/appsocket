# AppSocket

Lightweight, fast & simple WebSocket client

## Features

- Lightweight
- Fast / Performant
- Simple
- Responsive
- Session supported (only for this feature i did own app)
- LocalStorage saving
- Cross-browser (if you want use web-version)
